# Privacy Policy / 隐私政策

Last updated / 最近更新：2026-03-23

本隐私政策适用于 **Lumno | 聚焦搜索&新标签页**（下称“Lumno”或“本扩展”）。本页面说明当你使用由 Kubai087 发布的 Chrome 扩展程序时，Lumno 会如何处理相关信息。

## 1. 产品说明

Lumno 是一款浏览器效率工具，主要提供以下功能：

- 在网页中通过快捷键唤起搜索浮层
- 自定义新标签页
- 搜索和切换已打开标签页
- 搜索历史记录、书签和常用网站
- 支持站内搜索快捷词与搜索建议
- 在部分视频网站提供可选的画中画辅助能力

Lumno 的核心设计原则是：**默认尽量在本地浏览器内完成处理**。  
Lumno **不运行用于收集浏览历史、书签内容或网页标题的开发者后台服务**。

## 2. Lumno 可能访问的数据类型

根据你实际使用的功能，Lumno 可能访问以下类型的信息：

- **已打开的标签页**
  - 用于展示、排序、切换、恢复标签页。
- **浏览历史记录**
  - 用于本地历史搜索和排序。
- **书签**
  - 用于搜索书签，并在新标签页中展示书签快捷入口。
- **常用网站（Top Sites）**
  - 用于展示常访问网站。
- **网站内容**
  - 仅用于实现网页内浮层交互、快捷键响应，以及部分可选的画中画相关功能。
- **用户输入的搜索词**
  - 用于生成搜索结果、站内搜索结果以及可选的联网搜索建议。
- **扩展设置**
  - 例如主题、语言、布局偏好、搜索引擎偏好、站内搜索快捷词配置等。

## 3. 数据如何被处理

### 3.1 默认本地处理

以下处理默认在你的浏览器本地完成：

- 搜索与排序已打开标签页
- 搜索浏览历史
- 搜索书签
- 读取常用网站
- 渲染新标签页内容
- 保存扩展设置和部分本地缓存

Lumno **不会将你的浏览历史 URL、网页标题、书签树或标签页列表上传到开发者自有服务器**。

### 3.2 因功能触发的第三方请求

某些功能会让浏览器直接向第三方服务发起请求：

- **搜索建议**
  - 当你使用搜索建议功能时，你输入的查询词可能会发送给所选搜索引擎的建议接口，例如 Google、Bing、Baidu、Yahoo、360 Search、DuckDuckGo、Yandex、Quark、Shenma 或相关 Sogou 接口。
- **默认网页搜索**
  - 当你提交普通网页搜索时，查询词可能会发送给你选择的搜索引擎或浏览器默认搜索引擎。
- **网站图标（favicon）加载**
  - 为显示站点图标，Lumno 可能根据域名向第三方 favicon 服务或目标网站本身请求图标资源。

这些请求是由浏览器直接发往对应第三方服务，用于完成你主动使用的相关功能。

## 4. 本地存储

Lumno 使用 Chrome 扩展存储能力来保存设置和少量本地缓存，包括但不限于：

- `chrome.storage.local`
- `chrome.storage.sync`
- 在可用时使用的临时会话级扩展存储

这些存储主要用于：

- 主题与语言设置
- 搜索引擎偏好
- 书签与布局偏好
- 站内搜索快捷词配置
- 本地界面状态与轻量缓存

## 5. 权限用途说明

Lumno 可能申请以下权限，用途如下：

- **`tabs`**
  - 用于查询、切换、创建、更新和管理标签页。
- **`history`**
  - 用于提供本地历史记录搜索。
- **`bookmarks`**
  - 用于搜索与展示书签。
- **`topSites`**
  - 用于展示常访问网站。
- **`storage`**
  - 用于保存扩展设置。
- **`search`**
  - 用于调用浏览器支持的搜索能力。
- **`scripting`** 与页面访问权限（如 **`<all_urls>`**）
  - 用于在网页中注入扩展浮层、快捷键响应逻辑，以及支持可选的网页内功能。

## 6. 数据出售与转移

开发者声明，Lumno：

- **不会**出售用户数据
- **不会**将用户数据用于与扩展核心功能无关的目的
- **不会**将用户数据用于信用评估、放贷或类似用途

## 7. 儿童隐私

Lumno 并非面向 13 岁以下儿童设计。开发者不会通过自建后台服务主动收集儿童个人信息。

## 8. 安全说明

由于 Lumno 主要在本地处理数据，发送给开发者的数据范围有限。  
但如果你使用第三方搜索建议或第三方 favicon 服务，这些第三方可能会接收到你的 IP 地址、基础请求元数据、查询词或被请求的域名。

## 9. 政策更新

本隐私政策可能会不时更新。更新后的版本会发布在本页面，并同步调整“最近更新”日期。
